# ru-pr-description

Генерирует title и description для PR на русском по диффу ветки.

## Когда вызывается

- Явно: «напиши PR», «сгенерь PR description», «опиши что я делал в этой ветке».
- Автоматически: перед `gh pr create` без `--body`.

## Когда пропускается

- У PR уже есть описание.
- Черновик с явной меткой «wip».

## Использование

После `git push` (или при готовности создать PR):

1. Скажи Клоду: «напиши PR».
2. Он соберёт контекст (`git log`, `git diff` относительно `main`) и выдаст готовые title + body.
3. Скопируй в `gh pr create --title "..." --body "..."` или вставь в UI GitHub.

## Шаблон тела PR

```markdown
## TL;DR
<1–2 предложения>

## Что изменилось
- пункт

## Как тестировал
- pytest / npm test / ручные шаги

## Чек-лист
- [x] Тесты проходят
- [x] Линтер чист
- [ ] Changelog обновлён
- [ ] Документация обновлена
```
