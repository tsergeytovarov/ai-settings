# CHANGELOG — ru-commit-message

Формат: [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), семантическое версионирование.

## [1.0.0] — 2026-04-18

### Первый релиз
- Генерация conventional-commit message на русском по `git diff --staged`.
- Поддержка типов: `feat`, `fix`, `chore`, `refactor`, `docs`, `test`, `style`, `perf`, `ci`, `build`.
- Опциональный scope из путей изменённых файлов.
- Тело сообщения — на русском, объясняет ПОЧЕМУ.
