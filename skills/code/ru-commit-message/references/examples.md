# Примеры

## feat — новая функциональность

```
feat(api): добавить эндпоинт /users/me/preferences
```

```
feat(auth): поддержка magic-link логина

Альтернатива паролю — ссылка в email. Пароль остаётся дефолтом,
magic-link включается фичафлагом `magic_link_auth`.
```

## fix — исправление бага

```
fix(auth): снять устаревший JWT при смене пароля
```

```
fix(billing): корректно считать пропорциональный возврат при отмене

Было: возврат по полному тарифу. Стало: пропорционально оставшимся
дням подписки. Регрессионный тест добавлен.
```

## refactor — рефакторинг без изменения поведения

```
refactor(db): вынести сессию в зависимость вместо глобала
```

## docs — только документация

```
docs(readme): обновить секцию про установку uv
```

## test — только тесты

```
test(auth): добавить регрессионный тест на истекший токен
```

## chore — обслуживание, зависимости, конфиги

```
chore(deps): обновить FastAPI до 0.110
```

```
chore: настроить pre-commit hooks для ruff и mypy
```

## style — форматирование, без изменения логики

```
style: применить ruff format ко всему репозиторию
```

## perf — производительность

```
perf(api): кешировать результат list_users на 60 секунд
```

## ci / build

```
ci(github): добавить прогон skill-lint на каждый PR
```

```
build: обновить lockfile после bump зависимостей
```
