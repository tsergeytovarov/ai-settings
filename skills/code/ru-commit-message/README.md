# ru-commit-message

Скилл генерирует conventional-commit сообщение на русском по `git diff --staged`.

## Когда вызывается

- Явно: «напиши коммит», «сгенерь commit message», «опиши этот diff для коммита».
- Автоматически: перед `git commit` без `-m`.

## Когда пропускается

- Пользователь уже написал сообщение.
- Merge- или revert-коммиты (они пишутся автоматически).

## Использование

После `git add`:

1. Скажи Клоду: «напиши коммит».
2. Он прочитает `git diff --staged` и выдаст готовое сообщение.
3. Скопируй в `git commit -m "..."` или дай Клоду сразу закоммитить (с подтверждением).

## Пример

```
feat(auth): добавить вход через GitHub OAuth

Закрываем задачу #42. Обычный логин/пароль оставлен как fallback
на случай недоступности OAuth-провайдера.
```

Больше примеров по типам изменений — в `references/examples.md`.
