# tg-post-writer

Скилл пишет посты в Телеграм на русском в стиле Бориса — прямо, без воды, с одной мыслью на пост.

## Когда вызывается

- Явно: «напиши TG-пост про …», «оформи эту идею для телеграма», «сделай пост в тг».

## Когда пропускается

- Платформы, кроме Телеграма (Twitter, LinkedIn, Habr — отдельные скиллы).
- Формальные пресс-релизы.

## Использование

Дай Клоду идею, буллеты или чётный черновик. Скажи: «напиши TG-пост».

Если идей много — он переспросит, на чём сфокусироваться.

## Структура поста

- Хук (1–2 предложения) — зачем читать дальше.
- Тело (3–6 коротких абзацев) — одна мысль на абзац.
- Финал — вывод, вопрос читателю или call-to-action.

## Ограничения

- Длина: 500–1500 символов. Длиннее — структурный брейк или тред.
- Эмодзи: максимум 2 на пост, только со смыслом.
- Без «друзья, привет» и лести.
- Без дежурных списков на 5+ пунктов, если тема это не требует.
