# CHANGELOG — tg-post-writer

Формат: [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), семантическое версионирование.

## [1.1.0] — 2026-04-18

### Добавлено
- `references/style-guide.md` — полный стайл-гайд канала «··• Серёжа печатает»: голос, заходы, структура, ритм, лексика, правила кавычек, концовки, чек-лист перед сдачей (11 разделов).
- `references/examples.md` — 10 эталонных постов из канала по жанрам: новость + разбор, мысль / провокация, размышление / метанавык, короткая эмоция / самоирония.

### Изменено
- `SKILL.md` — добавлен блок «Required reading before writing» (style-guide + examples перед началом работы). Процесс расширен форматами из гайда, добавлены жёсткие правила (кавычки, стоп-лист заходов, запрещённые обороты) и чек-лист перед возвратом.
- `SKILL.md` description — уточнено, что целевой голос теперь — established voice канала, а не абстрактная «Boris persona».

## [1.0.0] — 2026-04-18

### Первый релиз
- Написание постов в Телеграм на русском в стиле персоны Бориса.
- Ограничения по длине (500–1500 символов) и форматированию под TG.
- Эмодзи — максимум 2 на пост, только со смыслом.
